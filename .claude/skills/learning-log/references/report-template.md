# 学習レポートテンプレート: learning-log

以下の形式で `03.Learnings/YYYY-MM-DD_{title}/REPORT.md` に出力する。

ファイル名ルール:
- `{title}` は学習テーマを短く表したもの（日本語可、英語可）
- ファイル名に使えない文字（`/`, `\`, `:`, `*`, `?`, `"`, `<`, `>`, `|`）は全角に変換する
- スペースはハイフン `-` に置換する

---

> 関連分野: {PROFILE.mdの該当分野}

## テンプレートの補足ルール

- **初学者レベルのわかりやすさ**を最重視する。専門用語には必ず括弧内で解説を付ける
- **Mermaid図は必ず1つ以上含める**。学習テーマに合わせて最適な図の種類を選択する
- NotebookLMの出力はそのまま使わず、対話の内容と統合してClaude自身がわかりやすく書き直す
- 英語のソースの場合、レポート本文は日本語で記述する（ファイル名は原文のまま可）
- 「実務への応用」セクションは PROFILE.md の情報を活用して具体的に記述する
- 「振り返り」セクションは対話で得られたユーザーの言葉をできるだけ反映する
- 全てのセクションを埋める必要はない。情報が不足している場合は該当セクションを省略する
- 「関連リンク」は学習ソース内で引用されている参考リンクを掲載する
