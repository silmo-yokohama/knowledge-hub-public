# マッチング評価基準

PROFILE.mdの興味領域に基づき、各記事を**カテゴリに分類**し、関連のない記事を除外する。

## 評価プロセス

1. 記事のタイトル（必要に応じて description）から、**何について書かれた記事か**を理解する
2. PROFILE.mdの興味領域のいずれかに該当するか判定する
3. 該当する場合 → 適切なカテゴリを割り当てる
4. どの興味領域にも該当しない場合 → レポートから除外する

## カテゴリ一覧

PROFILE.mdの興味領域から導出される主なカテゴリ:

| カテゴリ | 対応する興味領域 |
|---------|----------------|
| AI/LLM | AI/LLM全般、Claude Code、AI開発ツール、規制・社会影響 |
| フロントエンド | Next.js、Nuxt、UI/UX、JavaScript/TypeScriptエコシステム |
| バックエンド | Go、Python、PHP/Laravel、WordPress |
| DevOps/インフラ | DevOps、CI/CD、AWS/GCP/Azure |
| 設計/アーキテクチャ | クリーンアーキテクチャ、DDD、PM |
| キャリア/ビジネス | 個人開発、マネタイズ、起業、組織開発、税金、お金 |
| ベイスターズ | 横浜DeNAベイスターズの記事（選手、試合、ドラフト等） |
| 野球 | NPB/MLBの一般記事（他球団の話題、球界全体のニュース等） |
| 横浜エクセレンス | 横浜エクセレンスの記事（選手、試合等） |
| バスケ | Bリーグ/NBAの一般記事（他チームの話題、リーグ全体のニュース等） |
| スポーツ | その他スポーツ全般、オリンピック、ワールドカップ等の国際大会 |
| ゲーム | RPG、Co-op、サバイバル、サンドボックス |
| 生活 | 料理、節約、栄養 |
| 政治/経済 | 政治、選挙、経済 |
| 横浜・神奈川 | 横浜市・神奈川県のローカルニュース |

- 上記カテゴリに当てはまらない場合でも、PROFILE.mdの興味領域に関連する記事は適切なカテゴリ名を新設してよい
- 1つの記事は1つのカテゴリにのみ分類する（最も関連の強いカテゴリを選択）
- **カテゴリではなく記事の内容で判断する**（AI規制法案 → 「AI/LLM」、フリーランス新法 → 「キャリア/ビジネス」）
- **チーム固有の記事は専用カテゴリに分類する**:
  - 「DeNA」「ベイスターズ」「横浜DeNA」を含む記事 → 「ベイスターズ」
  - 「横浜エクセレンス」を含む記事 → 「横浜エクセレンス」
  - 横浜・神奈川のローカルニュース（特定チームの話題でない場合）→ 「横浜・神奈川」
  - NPB/MLBの一般的な話題（他球団、球界全体）→ 「野球」
  - Bリーグ/NBAの一般的な話題（他チーム、リーグ全体）→ 「バスケ」

## 除外ルール

以下の記事はレポートに含めない:
- PROFILE.mdのどの興味領域にも関連しない記事
  - ただし、ブックマーク数が500以上の記事は関連が薄くても含める（カテゴリ「注目」等を割り当てる）
- **公開日がレポート日付の3日以上前の記事**（例: レポート日が2/17なら、2/14以前の記事は除外）

## ソートルール

- 記事はカテゴリ別にグループ化する
- 各カテゴリ内では、スコア（はてブ数 / Redditポイント）の降順でソートする
- Yahooニュースはスコアがないため、各カテゴリの末尾に配置する

## 注意事項

- トークン節約のため、タイトルのみで判定する。判断困難な場合のみ description を参照する
